# 💰 Smart Expense Tracker

A modern and interactive **Expense Tracker Web Application** that helps users manage their finances by tracking **income, expenses, savings, and category-wise spending**.

The application provides a **clean dashboard UI, category analytics, and persistent data storage** so users can easily monitor their financial activity.

# ✨ Features

### 💰 Income Management

* Add multiple income sources such as:

  * Salary
  * Freelance
  * Side income
* Automatically updates total income and balance.

### 💸 Expense Tracking

Track expenses with categories like:

* Food
* Travel
* Bills
* Shopping
* Entertainment
* Health
* Education

### 📊 Financial Dashboard

Displays a real-time summary of:

* **Total Balance**
* **Total Income**
* **Total Expenses**
* **Savings**

### 📈 Expense Visualization

* Interactive **Pie Chart** using Chart.js
* Shows category-wise expense distribution.

### 🗂 Category Totals

Displays how much money was spent in each category.

### 🕒 Transaction History

Shows all income and expense records.

### 🗑 Delete Transactions

Users can remove income or expense entries.

### 🌙 Dark Mode

Toggle between light and dark themes.

### 💾 Save Data

Save financial data manually.

### 🔄 Reset Data

Clear all saved transactions instantly.

### 💽 Local Storage Support

All data is stored in the browser using **LocalStorage**, so refreshing the page **does not delete data**.
